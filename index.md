# Website Korwil Kalimantan Tengah — KAKP & KAKL

Website resmi Koordinator Wilayah Kalimantan Tengah untuk Komite Akreditasi
Kesehatan Pratama (KAKP) dan Komite Akreditasi Kesehatan Lanjutan (KAKL).

## Publikasi (GitHub Pages)
1. Repo ini berisi `index.html` — file tunggal siap tayang.
2. Aktifkan lewat Settings → Pages → Branch: `main` → folder `/ (root)`.
3. Website akan tersedia di:
   `https://<nama-organization>.github.io/<nama-repo>/`

## Status konten
Beberapa bagian masih placeholder dan perlu dilengkapi dengan data resmi
sebelum publikasi luas: statistik kegiatan, testimoni, dan tanggal artikel
berita.
